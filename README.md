マインドフルネスがコスモを生み出すのなら、その逆のカオスを生み出す装置を開発しようと。

「カオス理論」（「変数が少し変わるだけで結果が全く予想できなくなる」ような状態を指す）を用いてカオスの要素を紐解いていく。

・構成要素は日常に溢れている

・常に変化している

・不規則である

これらをヒントに、2つの方向性で映像と音楽を作成し、視覚、聴覚で体感してもらうためにこのサービスを開発した。
